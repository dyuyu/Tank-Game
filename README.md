# Tank-Game

**Blue Tank Controls:**
- Up arrow move right
- Down arrow move left
- Right arrow rotates clockwise
- Left arrow rotates the oposite direction
- M shoot bullets


**Pink Tank Controls:**
- W move right
- S down move left
- D rotates clockwise
- A Left arrow rotates the oposite direction
- Z shoot bullets
