# Tank-Game

**Blue Tank Controls:**
- Up arrow move right
- Down arrow move left
- Right arrow rotates clockwise
- Left arrow rotates counterclockwise
- M shoot bullets


**Pink Tank Controls:**
- W move right
- S down move left
- D rotates clockwise
- A rotates to the counterclockwise
- Z shoot bullets
