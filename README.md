# Tank-Game

**Blue Tank Controls:**
- Up arrow moves to the right
- Down arrow moves to the left
- Right arrow rotates clockwise
- Left arrow rotates counterclockwise
- M shoots bullets


**Pink Tank Controls:**
- W moves to the right
- S down moves to the left
- D rotates clockwise
- A rotates to the counterclockwise
- Z shoots bullets
